Guide — Connecter MetaTrader 5 au robot TORINVEST

Ce guide est réservé aux abonnés Robot Access. Il explique comment installer l’EA TORINVEST AI Executor v1.51, activer votre licence TOR-VIP-… et recevoir les signaux validés par le Worker Cloudflare TORINVEST.

Avant de commencer

Ce que tu dois recevoir de TORINVEST

Après paiement ou validation de ton abonnement Robot Access, tu reçois (email ou CRM) :

  • 1Une licence TOR-VIP-XXXX-XXXX-XXXX
  • 2Un code d’activation (si la licence n’est pas déjà liée à ton compte MT5)
  • 3Ton CopyToken personnel (jeton Worker — ne le partage jamais)
  • 4Le fichier EA TORINVEST AI Executor v1.51.ex5 (ou le source .mq5 à compiler)
⚠️ Le CopyToken n’est pas public sur le site. Ne le cherche pas sur Internet : il t’est transmis personnellement avec ta licence.
Étape 0

Activer la licence sur ton compte MT5

Si ton CRM a déjà renseigné ton numéro de compte MT5 à la création, passe à l’étape 1.

Sinon, active la licence avec ton numéro de compte MT5 et le code d’activation reçu.

Tu peux aussi vérifier l’état dans le terminal AI Access → panneau License Check.

💡 Connexion cockpit : /ai-access → onglet Client → colle ta licence TOR-VIP.
Étape 1

Installer MetaTrader 5

  • 1Ouvre un compte MT5 (démo ou réel) chez ton broker.
  • 2Installe MetaTrader 5 sur ton PC ou VPS (recommandé pour laisser l’EA tourner 24/5).
  • 3Connecte-toi avec identifiant, mot de passe et serveur broker.
Étape 2

Installer l’EA TORINVEST AI Executor v1.51

  • 1Dans MT5 : Fichier → Ouvrir le dossier des données.
  • 2Va dans MQL5 → Experts.
  • 3Copie TORINVEST AI Executor v1.51.ex5 (ou compile le .mq5 avec MetaEditor — F7).
  • 4Ferme MT5 complètement, puis relance-le.

Dans le Navigateur (Ctrl+N), tu dois voir l’EA sous Conseillers Experts.

Étape 3

Autoriser WebRequest (obligatoire)

MT5 bloque les appels HTTP par défaut. Autorise le Worker TORINVEST :

  • 1Outils → Options → Expert Advisors
  • 2Coche Autoriser le trading algorithmique
  • 3Coche Autoriser WebRequest pour les URL listées
  • 4Ajoute cette URL (une ligne) :
https://morning-hall-d8f6.onzerimes.workers.dev

Si tu utilises Chainwork (optionnel), ajoute aussi https://api.chainwork-ai.com.

Étape 4

Attacher l’EA au graphique

  • 1Ouvre un graphique de l’actif cible (ex. XAUUSD — vérifie le nom exact chez ton broker : Gold, XAUUSD, etc.).
  • 2Glisse TORINVEST AI Executor v1.51 sur le graphique.
  • 3Onglet Entrées : configure les paramètres ci-dessous.
  • 4Onglet Autorisations : coche Autoriser le trading algorithmique.
Étape 5

Paramètres EA (production 2026)

💡 L’ancien robot TorinvestCopier et l’URL ngrok ne sont plus utilisés. Utilise uniquement l’EA v1.51 et le Worker ci-dessus.
Étape 6

Activer Algo Trading

  • 1Le bouton Algo Trading / AutoTrading en haut de MT5 doit être VERT.
  • 2Un smiley sur le graphique (coin haut droit) — pas de croix rouge.
  • 3Onglet Experts du Journal : tu dois voir TORINVEST AI Executor v1.51 lancé.
✅ Si la licence est active et le Worker OK, l’EA poll /copy/signal toutes les ~30 s. Un message « aucun signal » est normal tant qu’aucun setup n’a été validé côté Forge.
Étape 7

Suivre le cockpit AI Access

L’EA exécute sur MT5. Le terminal web te donne le contexte (Smart Money, news, signaux, santé système) :

  • 1Connecte-toi à AI Access avec ta licence TOR-VIP.
  • 2Ouvre le terminal cockpit.
  • 3Vérifie Signal Center, System Health et License Check.
Dépannage

Problèmes fréquents

SymptômeCause probableAction
401 token / licence invalide CopyToken ou LICENSE_KEY incorrect Vérifie les valeurs reçues de TORINVEST (pas d’espace, licence active).
503 COPY_TOKEN Worker non configuré Contacte le support TORINVEST.
404 aucun signal Pas de signal en cache Normal hors session / sans setup validé — surveille le cockpit.
WebRequest failed URL non autorisée Ajoute morning-hall-d8f6.onzerimes.workers.dev dans Options MT5.
Licence pending Activation incomplète Active avec code + numéro compte MT5 ou contacte le CRM.
Rappel

Risque & contrôle

⚠️ Le trading automatisé comporte un risque de perte totale du capital. Teste d’abord sur compte démo avec un lot minimal (0,01).

Tu peux arrêter l’EA à tout moment : retire-le du graphique ou désactive Algo Trading (bouton rouge).
